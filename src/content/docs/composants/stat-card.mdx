Carte de statistique joueur/équipe — affiche une métrique clé (grand chiffre + glow, badge de tendance, sparkline) sur les dashboards et profils. **12 variants** sur 3 axes indépendants. Deux tailles : Default 268×128px et Compact 224×104px.

## Variants

| Axe      | Valeurs                   |
| -------- | ------------------------- |
| Tendance | Hausse · Baisse · Neutre  |
| Accent   | Lime · Violet             |
| Size     | Default · Compact         |

## Propriétés (instance overrides)

| Propriété        | Type    | Description                                      |
| ---------------- | ------- | ------------------------------------------------ |
| Valeur           | TEXT    | Grand chiffre affiché (ex. `47`, `68%`, `124h`)  |
| Libellé          | TEXT    | Nom de la métrique (ex. `MATCHS JOUÉS`)          |
| Delta            | TEXT    | Variation (ex. `+12%`, `-3`)                     |
| Tendance visible | BOOLEAN | Affiche/masque le badge de tendance              |
| Sparkline visible| BOOLEAN | Affiche/masque la barre dégradée en bas          |
| Icône visible    | BOOLEAN | Affiche/masque l'icône carré accent              |

## Anatomie

1. **Header** : icône optionnelle (carré accent teinté 16%) à gauche + badge Tendance à droite.
2. **Value** : SG Bold 40px (Default) / 28px (Compact), couleur accent + effet glow.
3. **Label** : SG Regular 12px (Default) / 11px (Compact), ink 60%, UPPERCASE.
4. **Sparkline** : rectangle en bas, 6px (Default) / 5px (Compact), gradient violet→accent.

## Mapping Tendance → visuel

| Tendance | Couleur badge   | Flèche | Delta   |
| -------- | --------------- | ------ | ------- |
| Hausse   | Accent          | ▲      | Positif |
| Baisse   | Rouge error     | ▼      | Négatif |
| Neutre   | ink 40%         | —      | `0`/`0%`|

## Props

```ts
value: string
label: string
delta?: string                              // masque le badge si absent
tendance?: 'hausse' | 'baisse' | 'neutre'  // défaut 'hausse'
accent?: 'lime' | 'violet'                 // défaut 'lime'
size?: 'default' | 'compact'               // défaut 'default'
showSparkline?: boolean                    // défaut true
showIcon?: boolean                         // défaut false
```

## Usage

**À faire**
- Grouper 3–4 Stat Cards en rangée dans les dashboards (grid gap 16px).
- Choisir `size="compact"` dans les espaces contraints.
- Toujours afficher un delta cohérent avec l'axe Tendance.
- Activer `Graph=true` uniquement pour la métrique principale.

**À éviter**
- Mettre plus d'une métrique par carte.
- Laisser `Tendance=Hausse` avec un delta négatif.

:::note[Adaptation light mode]
Dans le Figma (dark), le chiffre est en accent plein. Sur fond clair l'accent est illisible : on garde le **halo glow** comme signature mais le chiffre passe en **encre**. L'accent reste porté par la sparkline et le badge de tendance.
:::
