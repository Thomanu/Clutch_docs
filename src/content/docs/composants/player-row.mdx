## Props

```ts
name: string · avatar?: string
status?: 'online' | 'away' | 'offline'
role?: 'Duelist' | 'Initiator' | 'Controller' | 'Sentinel' | 'IGL' | 'Flex'
ready?: 'default' | 'ready' | 'notReady'
captain?, selected?, action?: boolean
statType?: 'kd' | 'rank' | 'winrate' | 'adr' | 'mvp' · statValue?: string
kdTrend?: 'none' | 'positive' | 'negative'
onAction?: () => void
```

## Usage

Le rôle, le statut et la stat sont composables. La barre `ready` (vert/rouge) signale l'état de préparation avant un match.
