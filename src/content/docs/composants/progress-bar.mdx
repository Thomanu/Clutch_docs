Barre de progression. Piste 4px (ink 10%), remplissage en **dégradé horizontal violet → lime**, radius 0. Utilisée dans Tournament Row (Z3 Teams) et les états de chargement.

## Aperçu

<img src="/images/composants/progress-bar.png" alt="Progress Bar — différentes valeurs et hauteurs" style="width:100%;border:1px solid rgba(26,25,28,.08)" />

## Specs

| Élément     | Valeur                                 |
| ----------- | -------------------------------------- |
| Piste       | hauteur 4px · fond ink 10% · radius 0  |
| Remplissage | dégradé violet #683EF6 → lime #C6FF33  |
| Largeur fill| proportionnelle à `value` (0–100)      |
| aria        | `role="progressbar"` · `aria-valuenow` |

## Props

```ts
value: number         // 0–100 (clampé automatiquement)
height?: number       // défaut 4
width?: number | string  // défaut '100%'
```

## Usage

Pour une progression déterminée (remplissage d'équipes, étapes, upload). La valeur est bornée automatiquement entre 0 et 100. Pour un loading indéterminé, utiliser le Spinner du composant Button.
