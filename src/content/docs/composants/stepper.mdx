Indicateur d'étapes (ex. création de tournoi). Carrés 36px (radius 0), étapes complétées/active en **lime** avec glow violet sur l'active. Connecteurs lime quand franchis.

## Aperçu

<img src="/images/composants/stepper.png" alt="Stepper — étapes 1, 2 et 3 actives" style="width:100%;border:1px solid rgba(26,25,28,.08)" />

## Specs

| Élément          | Valeur                                                       |
| ---------------- | ------------------------------------------------------------ |
| Layout           | colonne gap 8 · rangée de steps                              |
| Step (carré)     | 36×36px · radius 0                                           |
| Connecteur       | 2px × 48px                                                   |
| Complété / actif | fond lime · bordure lime · numéro ink ; actif = glow violet  |
| Inactif          | fond ink 4% · bordure ink 10% · numéro ink 60%               |
| Labels           | SG Bold 9px · UPPERCASE · largeur 36px · centrés             |

## Props

```ts
currentStep: number  // 1-based
steps: string[]      // labels des étapes
```

## Usage

- Étape `< currentStep` = complétée (lime plein).
- Étape `= currentStep` = active (lime + glow violet).
- Étape `> currentStep` = à faire (grisée).
