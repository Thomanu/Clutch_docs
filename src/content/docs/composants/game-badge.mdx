Badge identifiant le **jeu**. Police **IBM Plex Mono** (et non Space Grotesk), texte lime UPPERCASE, fond lime 10 %, bordure lime 40 %.

## Specs

| Propriété | Valeur                          |
| --------- | ------------------------------- |
| Padding   | H 8px · V 3px                   |
| Fond      | lime 10 % `rgba(198,255,51,.1)` |
| Bordure   | 1px lime 40 %                   |
| Texte     | IBM Plex Mono 9px · lime · UPPERCASE |
| Radius    | 0                               |

Variante `onImage` : overlay sombre `rgba(26,25,28,.6)`, bordure blanche 15 %, texte clair.

## Props

```ts
game: 'csgo' | 'r6' | 'valorant' | 'novacore'
onImage?: boolean
```

## Usage

Réservé à l'**identification d'un jeu**. À ne pas confondre avec le **Status Badge** (présence d'un joueur) ni le **State Badge** (état d'un objet).
