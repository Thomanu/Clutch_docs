Écran vide orné (coins HUD, glow). Code badge encadré, hero icon, titre hero, teaser de stats, double CTA. **Le CTA principal est violet** (arbitrage : violet = CTA), le secondaire en ghost.

## Specs

| Élément      | Valeur                                                    |
| ------------ | --------------------------------------------------------- |
| Conteneur    | col · gap 32 · p40 · w560 · centré                        |
| Code badge   | filet + eyebrow SG Bold 11px (tracking 1.65) + filet      |
| Hero icon    | 120px · tint d'accent + glow + 4 coins HUD                |
| Titre        | SG Bold 40px · tracking 2px                               |
| Stats teaser | 3 boîtes · valeur SG Bold 24px (violet / lime+glow / ink) |
| CTA          | **primaire violet** + secondaire ghost                    |
| Hint         | filet + texte SG 13px + filet                             |

## Props

```ts
eyebrow: string
title: string
description: string
icon?: string
accent?: 'error' | 'violet' | 'lime'
stats?: { value: string; label: string; accent?: string }[]
primaryCta: { label: string; onClick?: () => void }
secondaryCta?: { label: string; onClick?: () => void }
hint?: string
```

## Usage

`accent` colore le code badge et le hero icon — rouge pour un vide « bloquant », lime pour un vide « positif » (ex. aucun résultat mais invitation à créer), violet pour un vide neutre.
