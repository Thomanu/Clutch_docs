Carte d'événement / scrim. 548×88px, bloc date + contenu + zone de slots d'équipes (5 carrés). État `live` = bordure rouge, `past` = opacité réduite.

## Aperçu live

  <EventCard state="past" date={{ weekday: 'Lun', day: 15 }} title="Scrim terminé" meta="Joué · 4/5" slotsFilled={4} client:load />
</div>

## Specs

| Propriété      | Valeur                                            |
| -------------- | ------------------------------------------------- |
| Dimensions     | 548×88px · padding 16 · gap 16                     |
| Bordure        | open/full/past 1px ink 10 % · **live 1px rouge**  |
| Past           | + opacity 0.55                                    |
| Date block     | mono 10px (jour) + SG Bold 24px (n°)              |
| Slots          | 5 carrés 10px (lime si remplis ; vide = rouge teinté en open) |

## Props

```ts
state?: 'open' | 'full' | 'live' | 'past'
date: { weekday: string; day: string | number }
title: string · meta: string
slotsFilled: number · slotsTotal?: number  // défaut 5
onClick?: () => void
```

## Usage

Liste d'événements/scrims à venir. Le bloc date utilise la police mono (technique). `live` attire l'œil par sa bordure rouge.
