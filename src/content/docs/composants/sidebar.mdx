## Spécifications clés

- Largeur **256px** (Expanded) · fond `clutch-color-contextual-100` · bordure droite 1px `clutch-color-primary-20`.
- **Logo CLUTCH** : Alexandria Black, underline gradient violet, rotation −4.5°.
- **Items de menu** : icône 24px + `Body/M/Regular` · padding 12px H · séparateur `primary-20`.
- **Auth=LoggedOut** : le bloc profil est remplacé par un CTA de connexion.

## Usage

**À faire**
- Toujours appliquer le bon `State` pour marquer la page active dans la navigation.
- Utiliser `Auth=LoggedOut` sur les écrans publics (landing, connexion).
- `Size=Collapsed` pour les vues où la navigation doit s'effacer au profit du contenu.

**À éviter**
- Modifier l'ordre des items de navigation sur un écran particulier — il est global et cohérent.
- Détacher l'instance ou modifier la Sidebar directement depuis un écran maquette.
- Éditer le doublon de la page « Design System » (supprimé — seul `2202:2` est canonique).
