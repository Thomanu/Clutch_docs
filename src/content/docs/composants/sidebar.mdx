## Variants

| Axe   | Valeurs                                                                       |
| ----- | ----------------------------------------------------------------------------- |
| State | default · dashboard · squadhub · tacticalboard · arena · monprofil · notifications · aide |
| Size  | expanded (256px) · collapsed (64px)                                           |
| Auth  | loggedIn · loggedOut                                                          |
| Theme | purple · lime                                                                 |

## Props

```ts
activePage?: 'default' | 'dashboard' | 'squadhub' | 'tacticalboard'
           | 'arena' | 'monprofil' | 'notifications' | 'aide'
theme?: 'purple' | 'lime'        // défaut 'purple'
size?: 'expanded' | 'collapsed'  // défaut 'expanded'
auth?: 'loggedIn' | 'loggedOut'  // défaut 'loggedIn'
username?: string
userTag?: string
nextMatch?: { opponent: string; time: string; game: string }
serverName?: string
```

## Usage

- Toujours appliquer le bon `activePage` pour marquer la page active dans la navigation.
- Utiliser `auth="loggedOut"` sur les écrans publics (landing, connexion).
- `size="collapsed"` pour les vues où la navigation doit s'effacer au profit du contenu.
- Ne pas modifier l'ordre des items de navigation par écran — il est global et cohérent.
