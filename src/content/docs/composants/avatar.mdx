## Specs

| Size | Dimensions | Radius |
| ---- | ---------- | ------ |
| sm   | 24×24px    | 0      |
| md   | 32×32px    | 0      |
| lg   | 40×40px    | 0      |
| xl   | 64×64px    | 0      |

Bordure 1px ink 40 %. Pastille de statut 10px (sur L) avec contour blanc.

## Props

```ts
size?: 'sm' | 'md' | 'lg' | 'xl'   // défaut 'md'
src?, alt?: string
status?: 'online' | 'away' | 'offline'
```

## Usage

Sans `src`, l'avatar affiche les initiales déduites de `alt`. La pastille reprend la couleur du **Status Badge** correspondant.
