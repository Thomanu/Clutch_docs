Indique l'**état d'un objet** (tournoi, match, inscription). Texte 10px UPPERCASE, bordure de la couleur de l'état, fond teinté. Radius forcé à 0.

## Aperçu

<img src="/images/composants/badge.png" alt="Badges — Game Badge, Status Badge, State Badge" style="width:100%;border:1px solid rgba(26,25,28,.08)" />

## Specs

| Propriété | Valeur                              |
| --------- | ----------------------------------- |
| Padding   | H 10px · V 4px                      |
| Texte     | Space Grotesk Bold 10px · UPPERCASE |
| Bordure   | 1px (couleur de l'état)             |
| Radius    | 0                                   |

| État      | Couleur          |
| --------- | ---------------- |
| ouvert    | success #388e3c  |
| complet   | warning #f5a524  |
| live      | error #fe3939    |
| ferme     | ink 40%          |
| enAttente | warning #f5a524  |
| confirme  | success #388e3c  |
| enCours   | lime #C6FF33     |
| aVenir    | ink 40%          |

## Props

```ts
state: 'ouvert' | 'complet' | 'live' | 'ferme'
     | 'enAttente' | 'confirme' | 'enCours' | 'aVenir'
label?: string
```

## Usage

Pour l'état d'un **objet** (≠ Status Badge qui concerne la présence d'un joueur). Un seul vert (`#388e3c`) et un seul rouge (`#fe3939`) dans tout le système.
