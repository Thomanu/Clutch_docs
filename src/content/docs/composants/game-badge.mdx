Badge identifiant le **jeu**. Police **IBM Plex Mono** (et non Space Grotesk), texte UPPERCASE, fond lime 10%, bordure lime 40%. Variante `onImage` pour les overlays sur photo.

## Aperçu

<img src="/images/composants/badge.png" alt="Badges — Game Badge, Status Badge, State Badge" style="width:100%;border:1px solid rgba(26,25,28,.08)" />

## Specs

| Propriété | Valeur                            |
| --------- | --------------------------------- |
| Padding   | H 8px · V 3px                     |
| Fond      | lime 10% `rgba(198,255,51,.1)`    |
| Bordure   | 1px lime 40%                      |
| Texte     | IBM Plex Mono 9px · ink · UPPERCASE |
| Radius    | 0                                 |

Variante `onImage` : overlay sombre `rgba(26,25,28,.6)`, bordure blanche 15%, texte snow.

## Props

```ts
game: 'csgo' | 'r6' | 'valorant' | 'novacore'
onImage?: boolean
```

## Usage

Réservé à l'**identification d'un jeu**. À ne pas confondre avec le **Status Badge** (présence d'un joueur) ni le **State Badge** (état d'un objet).
