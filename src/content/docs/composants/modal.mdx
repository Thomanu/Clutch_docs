Fenêtre modale. 3 types : **confirmation** (coins HUD violet), **destructive** (bordure rouge + zone danger), **succès** (anneau lime + coins HUD lime). Overlay noir 70%.

## Aperçu

<img src="/images/composants/modal.png" alt="Modal — types confirmation, destructive et succès" style="width:100%;border:1px solid rgba(255,253,250,.08)" />

## Specs

| Type         | Conteneur                                                      |
| ------------ | -------------------------------------------------------------- |
| Confirmation | 520px · coins HUD violet (TL/TR) · confirmer violet            |
| Destructive  | 480px · bordure rouge + glow · zone danger + conséquences      |
| Succès       | 480px · anneau lime + coins HUD lime · confirmer lime          |

Tous partagent : overlay noir 70%, fond surface, header + footer, bouton de fermeture.

## Props

```ts
type?: 'confirmation' | 'destructive' | 'success'
title: string
body?: string
highlight?: string       // pill équipe (confirmation)
consequences?: string[]  // liste (destructive)
confirmLabel?: string
cancelLabel?: string
open?: boolean
inline?: boolean         // affiche le panneau dans le flux (usage docs)
onConfirm?: () => void
onClose?: () => void
```

## Usage

- **Destructive** pour les actions irréversibles — toujours lister les conséquences.
- Un seul Modal actif à la fois par vue.
- Le bouton positif (succès) est en **lime avec glow**.
- Fermeture via le bouton ✕, le bouton annuler, ou un clic sur l'overlay.
