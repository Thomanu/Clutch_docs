## Specs

| Axe      | Valeurs                          |
| -------- | -------------------------------- |
| Tendance | Hausse · Baisse · Neutre         |
| Accent   | Lime · Violet                    |
| Size     | Default (268×128px) · Compact (224×104px) |

| Élément     | Default                      | Compact                      |
| ----------- | ---------------------------- | ---------------------------- |
| Dimensions  | 268×128px                    | 224×104px                    |
| Padding     | 20px t/l/r · 16px b          | 16px t/l/r · 12px b          |
| Valeur      | SG Bold **40px**             | SG Bold **28px**             |
| Label       | SG Regular **12px** UPPERCASE | SG Regular **11px** UPPERCASE |
| Sparkline   | **6px** gradient violet→accent | **5px** gradient violet→accent |
| Trend badge | fond teinté 16% · bordure accent | idem |

## Mapping Tendance

| Tendance | Couleur badge | Flèche | Delta     |
| -------- | ------------- | ------ | --------- |
| Hausse   | Accent        | ▲      | Positif   |
| Baisse   | Rouge error   | ▼      | Négatif   |
| Neutre   | ink 40%       | —      | `0`/`0%`  |

## Props

```ts
value: string
label: string
delta?: string               // masque le badge si absent
tendance?: 'hausse' | 'baisse' | 'neutre'   // défaut 'hausse'
accent?: 'lime' | 'violet'                   // défaut 'lime'
size?: 'default' | 'compact'                 // défaut 'default'
showSparkline?: boolean                       // défaut true
showIcon?: boolean                            // icône carré accent en haut gauche
```

## Usage

- Grouper 3–4 Stat Cards en rangée dans les dashboards (grid gap 16px).
- Choisir `size="compact"` dans les espaces contraints.
- Toujours afficher un delta cohérent avec l'axe Tendance.

:::note[Adaptation light mode]
Dans le Figma (dark), le chiffre est en **accent plein**. Sur fond clair l'accent est illisible : on garde le **halo glow** comme signature mais le chiffre passe en **encre**. L'accent reste porté par la sparkline et le badge de tendance.
:::
