Ligne de joueur de Clutch — affiche un membre du roster : avatar (anneau coloré + pastille de présence), pseudo, rôle (tag coloré) et stat clé. **22 variants** sur 9 axes indépendants. Deux formats : **Compact** (548×64px) et **Table** (1120×72px, 8 colonnes).

## Variants

| Axe      | Valeurs                                                 |
| -------- | ------------------------------------------------------- |
| Status   | Online · Away · Offline                                 |
| Action   | None · Action                                           |
| Role     | Duelist · Initiator · Controller · Sentinel · IGL · Flex |
| Ready    | Default · Ready · NotReady                              |
| Captain  | None · Captain                                          |
| Selected | Default · Selected                                      |
| Stat     | KD · Rank · Winrate · ADR · MVP                         |
| KD       | Default · Positive · Negative                           |
| Layout   | Compact · Table                                         |

## Mapping rôle → couleur tag

| Rôle              | Couleur tag                    |
| ----------------- | ------------------------------ |
| Duelist           | Rouge `error`                  |
| Initiator · IGL   | Lime `lime-100`                |
| Controller        | Vert `success`                 |
| Sentinel · Flex   | Neutre `primary-40`            |
| (état Selected)   | Violet — réservé interaction   |

## Anatomie — Layout Compact (548×64px)

- **Avatar** : anneau secondary 30% opacity + pastille présence 10×10px.
- **Name Block** : pseudo SG Bold 14px + tag rôle coloré (fond 10% teinté + bordure).
- **Badge Captain « C »** : puce 16×16px lime avec texte ink.
- **Stat** : valeur clé à droite (IBM Plex Mono, lime/rouge selon KD).

## Anatomie — Layout Table (1120×72px)

| Col | Contenu                                     |
| --- | ------------------------------------------- |
| 1   | Avatar + pastille de présence               |
| 2   | Pseudo                                      |
| 3   | Ancienneté (`Depuis X mois`)                |
| 4   | Tag rôle                                    |
| 5   | Disponibilité (7 carrés L M M J V S D)      |
| 6   | K/D                                         |
| 7   | Matchs joués                                |
| 8   | Win Rate (barre dégradée + %) + Statut badge |


## Usage

**À faire**
- `Layout=Compact` dans les sidebars, listes de roster, slots d'équipe.
- `Layout=Table` dans les vues listing de squad hub (largeur 1120px).
- `Captain=Captain` sur exactement un joueur par roster.
- Choisir l'axe `Stat` pertinent au contexte (KD pour arène, Winrate pour profil).

**À éviter**
- Combiner `Status=Offline` + `Ready=Ready` (incohérence sémantique).
- Afficher plus d'une stat clé par ligne.
- Mixer Layout=Compact et Layout=Table dans la même liste.
