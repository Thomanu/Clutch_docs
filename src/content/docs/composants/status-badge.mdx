Indique la **présence d'un joueur**. Pastille colorée 8px + bordure de la couleur du statut + fond teinté. Le texte reste en encre pour rester lisible en light mode.

## Aperçu

<img src="/images/composants/badge.png" alt="Badges — Game Badge, Status Badge, State Badge" style="width:100%;border:1px solid rgba(26,25,28,.08)" />

## Specs

| Propriété | Valeur                      |
| --------- | --------------------------- |
| Padding   | H 8px · V 4px · gap 6px     |
| Pastille  | 8×8px · radius 0            |
| Texte     | Space Grotesk Bold 12px     |
| Bordure   | 1px (couleur du statut)     |
| Radius    | 0                           |

| Statut  | Couleur         | Fond          |
| ------- | --------------- | ------------- |
| online  | lime #C6FF33    | lime ~8%      |
| away    | warning #f5a524 | warning ~10%  |
| offline | ink 40%         | ink ~4%       |

## Props

```ts
status: 'online' | 'away' | 'offline'
label?: string  // défaut = libellé FR du statut
```

## Usage

Sert à la **présence** uniquement. La pastille de statut se retrouve aussi en overlay sur l'**Avatar** (composant dédié).
