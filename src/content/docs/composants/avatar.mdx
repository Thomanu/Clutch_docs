Photo de profil. **Carré, jamais circulaire** (radius 0) — arbitrage brutalisme Clutch. Pastille de statut optionnelle en overlay bas-droite. Sans `src`, affiche les initiales déduites de `alt`.

## Aperçu

## Specs

| Size | Dimensions | Radius |
| ---- | ---------- | ------ |
| sm   | 24×24px    | 0      |
| md   | 32×32px    | 0      |
| lg   | 40×40px    | 0      |
| xl   | 64×64px    | 0      |

- Bordure 1px ink 40%.
- Pastille de statut 10×10px · overlay bas-droite · contour 2px blanc.
- Couleurs pastille : online = lime · away = warning · offline = ink 40%.

## Props

```ts
size?: 'sm' | 'md' | 'lg' | 'xl'  // défaut 'md'
src?: string
alt?: string
status?: 'online' | 'away' | 'offline'
```

## Usage

La pastille reprend la couleur du **Status Badge** correspondant. Ne jamais appliquer un `border-radius` — le carré est une règle absolue du DS Clutch.
