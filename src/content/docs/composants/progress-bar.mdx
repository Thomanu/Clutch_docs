## Specs

| Élément      | Valeur                                   |
| ------------ | ---------------------------------------- |
| Piste        | hauteur 4px · fond ink 10 % · radius 0    |
| Remplissage  | dégradé violet #683EF6 → lime #C6FF33     |
| Largeur fill | = % de progression                       |

## Props

```ts
value: number      // 0–100 (clampé)
height?: number    // défaut 4
width?: number | string  // défaut '100%'
```

## Usage

Pour une progression déterminée (remplissage d'équipes, étapes, upload). La valeur est bornée automatiquement entre 0 et 100.
