Le bouton est l'élément d'action principal. **Le violet `#683EF6` est réservé au CTA principal** (variante `primary`). Le lime `#C6FF33` (variante `secondary`) est l'accent compétitif / succès. `borderRadius: 0` partout, texte en Space Grotesk Bold.

## Props

```ts
variant?: 'primary' | 'secondary' | 'tertiary' | 'destructive' | 'ghost' // défaut 'primary'
size?: 'sm' | 'md' | 'lg'                                                // défaut 'md'
disabled?: boolean
loading?: boolean
icon?: IconName | ReactNode   // 'arrow' | 'plus' | 'check' | 'trash' | 'search'
iconPosition?: 'left' | 'right'
iconOnly?: boolean
onClick?: (e) => void
```

## Usage

**À faire**
- Un seul CTA `primary` (violet) par vue — c'est l'action dominante.
- `secondary` (lime) pour une action de succès / validation compétitive.
- `destructive` pour les actions irréversibles (supprimer, quitter).
- `ghost` pour les actions secondaires neutres, `tertiary` pour un fond subtil.
- `iconOnly` uniquement si l'action est évidente — toujours fournir un `aria-label`.

**À éviter**
- Plusieurs CTA violets en concurrence sur le même écran.
- Utiliser le violet pour un état informatif (réservé CTA — règle absolue Clutch).
- Libellés longs : viser un verbe d'action court (« Créer un tournoi »).
