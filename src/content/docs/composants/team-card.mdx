Carte d'équipe. 544×156px, header (nom + badge d'état), info jeu/membres, footer bouton. Radius forcé à 0 (Figma : 4px).

## Specs

| Élément   | Valeur                                                |
| --------- | ----------------------------------------------------- |
| Conteneur | 544×156px · fond ink 4 % · bordure 1px ink 10 % · radius 0 |
| Header    | nom SG Bold 18 + badge d'état (px10 py4)              |
| Badge     | ouvert = success #388e3c · fermé = ink 40 %           |
| Info      | SG 14 · « jeu · current/max membres »                 |
| Footer    | bouton « Voir l'équipe » SG Medium 14                 |

Le vert `#33c766` du Figma est harmonisé vers `#388e3c` (success unique).

## Props

```ts
state?: 'ouvert' | 'ferme'
name: string · game: string
members: { current: number; max: number }
onView?: () => void
```

## Usage

Pour présenter une équipe dans une grille. L'état `ouvert` (recrutement) utilise le vert succès harmonisé.
