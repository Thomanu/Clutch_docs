Indique la **présence d'un joueur**. Pastille colorée 8px + bordure de la couleur du statut + fond teinté ; le texte reste en encre pour rester lisible en light mode.

## Aperçu live

<div style={{ display: 'flex', flexWrap: 'wrap', gap: '12px', alignItems: 'center', padding: '24px', background: '#f6f6f8' }}>
  <StatusBadge status="online" client:load />
  <StatusBadge status="away" client:load />
  <StatusBadge status="offline" client:load />
</div>

## Specs

| Propriété | Valeur                       |
| --------- | ---------------------------- |
| Padding   | H 8px · V 4px · gap 6px       |
| Pastille  | 8×8px                        |
| Texte     | Space Grotesk Bold 12/25      |
| Bordure   | 1px (couleur du statut)       |
| Radius    | 0                            |

| Statut    | Couleur            | Fond           |
| --------- | ------------------ | -------------- |
| online    | lime #C6FF33       | lime ~8 %      |
| away      | warning #f5a524    | warning ~10 %  |
| offline   | ink 40 %           | ink ~4 %       |

## Props

```ts
status: 'online' | 'away' | 'offline'
label?: string  // défaut = libellé FR du statut
```

## Usage

Sert à la **présence** uniquement. La pastille de statut se retrouve aussi en overlay sur l'**Avatar**.
