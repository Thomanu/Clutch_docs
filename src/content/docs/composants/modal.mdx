## Specs

| Type         | Conteneur                                            |
| ------------ | ---------------------------------------------------- |
| Confirmation | 520px · coins HUD violet (TL/TR) · confirm violet     |
| Destructive  | 480px · bordure rouge + glow · zone danger + conséquences |
| Succès       | 480px · anneau lime + coins HUD lime · confirm lime   |

Tous partagent : overlay noir 70 %, fond #fff (remap de #1f1e22), header + footer, bouton de fermeture.

## Props

```ts
type?: 'confirmation' | 'destructive' | 'success'
title: string · body?: string
highlight?: string          // pill équipe (confirmation)
consequences?: string[]     // (destructive)
confirmLabel?, cancelLabel?: string
open?: boolean · inline?: boolean
onConfirm?, onClose?: () => void
```

## Usage

- `inline` (doc) affiche le panneau dans le flux ; sinon le composant est un overlay piloté par `open`.
- **Destructive** pour les actions irréversibles — toujours lister les conséquences.
- Le bouton « positif » (succès) est en **lime avec glow**.
