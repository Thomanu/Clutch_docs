| État   | Badge   | CTA              |
| ------ | ------- | ---------------- |
| open   | OUVERT  | Rejoindre (violet)|

| closed | FERMÉ   | Résultats         |

## Props

```ts
state?: 'open' | 'full' | 'live' | 'closed'
game · title · organizer: string
details: { format, date, prize, maps }
teams: { filled: number; total: number }
onAction?: () => void
```

## Usage

Composant d'organisme : assemble Game Badge, State Badge, Progress Bar et Button. La largeur s'adapte (max 1120px).
