Cellule de match d'un bracket. Deux slots (équipe + score) séparés par un filet. Gagnant = teinte lime + accent gauche, perdant = teinte rouge atténuée. État `live` = pill rouge + slots violet.

## Aperçu

<img src="/images/composants/bracket.png" alt="Bracket Match — états default et live" style="width:100%;border:1px solid rgba(255,253,250,.08)" />

## Specs

| Élément         | Valeur                                                       |
| --------------- | ------------------------------------------------------------ |
| Slot            | 220×52px · px14 · gap10 · nom SG Medium 13 · score SG Bold 18 |
| Séparateur      | 1px ink 5%                                                   |
| Gagnant (default)| fond lime 10% · accent gauche 3px lime                      |
| Perdant (default)| fond error 10% · opacity 0.55 · score rouge                 |
| Pill live       | fond `#fe3939` · point 5px · « LIVE »                        |
| Slots live      | fond violet 8% · bordure violet 30% · scores encre          |

## Props

```ts
state?: 'default' | 'live'
teams: [
  { name: string; score: number; winner?: boolean },
  { name: string; score: number; winner?: boolean }
]
```

## Usage

Brique d'un arbre de bracket. `winner: true` colore le slot gagnant en lime. `live` signale le match en cours avec la pill rouge et les slots violet.
